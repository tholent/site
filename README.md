# Tholent

Finding thoughtful solutions to complex problems.

Software studio and consultancy — [tholent.com](https://tholent.com)

## Development

```bash
bundle install
bundle exec jekyll serve
```

## Structure

- `_layouts/` — default, home, page, post
- `_posts/` — blog posts (YYYY-MM-DD-title.md)
- `assets/main.scss` — full stylesheet
- `_config.yml` — site configuration
